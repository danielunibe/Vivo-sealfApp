# Investigación de Modelos y Assets de Dispositivos (Fase 2)

Este documento recopila la investigación de especificaciones comerciales, variantes, colores oficiales y estrategia de activos visuales para los modelos de teléfonos de la marca **vivo** representados en la aplicación **Vivo Promotor**.

---

## 1. vivo Y04

### **Detalles Comerciales**
*   **Nombre Comercial Exacto:** vivo Y04
*   **Variantes:** Modelo base 4G/5G con procesador de entrada.
*   **Colores Oficiales (Comerciales):**
    *   **Crystal Violet / Lavanda Cristal** (Morado claro pastel con textura cristalina)
    *   **Jade Green / Negro Jade** (Verde muy oscuro, casi negro satinado)
    *   **Titanium Gold** (Dorado metálico premium)

### **Especificaciones Clave**
*   **Pantalla:** 6.74 pulgadas, IPS LCD, 90Hz, resolución HD+ (720 x 1600 px).
*   **Procesador:** Unisoc T7225 (12nm).
*   **Memoria:** 4GB RAM + 64GB / 128GB / 256GB Almacenamiento (expandible a 2TB).
*   **Batería:** 5500 mAh.
*   **Carga:** 15W.
*   **Cámaras:** Trasera dual con sensor principal de 13 MP; Frontal de 5 MP.
*   **Durabilidad:** IP64 (resistencia a salpicaduras) y resistencia militar a caídas MIL-STD-810H.

### **Estrategia de Assets**
*   **Rutas Esperadas:**
    *   `public/assets/devices/y04/register.png` (vista vertical frontal/trasera con fondo transparente para carrusel de venta).
    *   `public/assets/devices/y04/thumb.png` (miniatura optimizada de 150x150px).
    *   `public/assets/devices/y04/catalog.png` (render 3/4 o inclinado para catálogo).
*   **Recomendación de Asset:** Render de prensa oficial en color Lavanda Cristal o Titanium Gold, recortado con fondo transparente en formato WebP/PNG para óptimo rendimiento móvil.
